# BPO Management System

A full-stack BPO Management Dashboard built using React, Flask, SQLite, and Tailwind CSS.
This project helps manage employees, projects, tickets, attendance, reports, and overall business workflow in a centralized dashboard.

---

# 🚀 Features

* Employee Management (CRUD)
* Dynamic Dashboard
* Login Authentication
* Attendance Module
* Reports & Analytics
* Projects Management
* Tickets Management
* Responsive Mobile-Friendly UI
* Flask REST API Integration
* SQLite Database Integration

---

# 🛠️ Tech Stack

## Frontend

* React.js
* Vite
* Tailwind CSS

## Backend

* Flask
* Flask-CORS
* SQLAlchemy

## Database

* SQLite

# ⚙️ Installation & Setup

## 1. Clone Repository

```bash
git clone https://github.com/jahnvisrivastava01/bpo-management-system.git
```

---

## 2. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```plaintext
http://localhost:5173
```

---

## 3. Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

Backend runs on:

```plaintext
http://127.0.0.1:5000
```
